# 🔍 MailXtract v4.2 — Email OSINT Tool

Advanced Email Investigation Tool — Domain Recon, WHOIS, DNS, Geolocation & More

## Features
- Email validation (RFC format check)
- Disposable email detection
- WHOIS lookup (registrar, creation date, expiry)
- IP address & geolocation (city, region, country)
- ISP detection
- DNSSEC check
- Blacklist check (Spamhaus, Barracuda)
- SPF & DMARC record extraction
- MX record lookup
- Gravatar check
- Web UI (localhost:8080)
- TXT, HTML & JSON report export
